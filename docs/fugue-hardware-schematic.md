# Fugue Hardware Schematic

Full schematic for the Daisy Seed-based Eurorack Fugue module. Each subsystem is shown with pin-level connections.

## 1. Power Supply

```

        │

        │

        │
```


### CV Input Conditioning Circuit (one per CV input, 7 total)

```
                              3.3V
                               │
                             [150k]  R2
                               │
  JACK TIP ──[100k]──┬────────┤────────> to CD4051 input
              R1      │        │
                    [BAT54S]   │
                   ┌──┤──┐     │
                   │  │  │     │
                 3.3V │ GND    │
                      │      [100nF]  C (optional filter)
                      │        │
                     GND      GND

VOLTAGE MAPPING:
  Vin = -5V  -->  Vout = (-5 * 150k + 3.3 * 100k) / (100k + 150k)
                       = (-750 + 330) / 250 = -1.68V --> clamped to ~0V by diode

  Vin = 0V   -->  Vout = (0 + 3.3 * 100k) / (100k + 150k)
                       = 330 / 250 = 1.32V

  Vin = +5V  -->  Vout = (5 * 150k + 3.3 * 100k) / (100k + 150k)
                       = (750 + 330) / 250 = 4.32V --> clamped to ~3.3V by diode

  Effective linear range after clamping: approximately -3V to +5V maps to 0V to 3.3V
  Firmware applies inverse mapping to recover original voltage.

DIODE DETAIL (BAT54S, SOT-23):
  Pin 1 (common cathode) ──── to signal node
  Pin 2 (anode 1) ─────────── GND
  Pin 3 (anode 2) ─────────── 3.3V

  Clamps signal to GND - 0.3V ... 3.3V + 0.3V
```

## 5. Clock and Reset Input Conditioning

```
CLOCK A INPUT (repeat for Clock B, Clock C, Reset)

  JACK TIP ──[100k]──┬──────────> Daisy GPIO (D12/D13/D14/D15)
              R1      │            (internal pulldown enabled in firmware)
                    [BAT54S]
                   ┌──┤──┐
                   │  │  │
                 3.3V │ GND
                      │
                   [100nF]  C (filter)
                      │
                     GND

MAPPING:
  Clock A .... D12
  Clock B .... D13
  Clock C .... D14
  Reset   .... D15

TRIGGER THRESHOLD:
  Daisy GPIO Schmitt trigger input.
  100k divider drops Eurorack +5V to ~1.6V, +10V to ~3.1V.
  Both cross the 3.3V logic high threshold after divider.

  Note: The 100k input resistor with the GPIO internal pulldown
  forms a voltage divider. For +5V Eurorack triggers:
  Vgpio = 5V * Rpulldown / (100k + Rpulldown)
  With ~50k internal pulldown: Vgpio = 5 * 50/150 = 1.67V
  Above the ~1.2V Schmitt trigger threshold.
```

## 6. Reset Button

```
  3.3V
   │
 [10k]  R (pullup)
   │
   ├──────────> Daisy D16 (input, no internal pull)
   │
 [BUTTON]  (momentary, normally open)
   │
  GND

  Idle: D16 reads HIGH (3.3V through pullup)
  Pressed: D16 reads LOW (shorted to GND)
  Firmware detects falling edge as trigger.
```

```
                 ┌────────────────┐        ┌────────────────┐        ┌────────────────┐

  Daisy D6 ────> │ 2 CLK      15│──GND  > │ 2 CLK      15│──GND  > │ 2 CLK      15│──GND

                 │ 4 D5          │         │ 4 D5          │         │ 4 D5          │
                 │ 5 D6          │         │ 5 D6          │         │ 5 D6          │
                 │ 6 D7          │         │ 6 D7          │         │ 6 D7          │
                 │ 7 QH'         │         │ 7 QH'         │         │ 7 QH'         │
                 │ 8 GND         │         │ 8 GND         │         │ 8 GND         │
  Daisy D7 <──── │ 9 QH       14│D3       │ 9 QH ──────>10│SER   14│D3              │
                 │10 SER ──GND   │         │10 SER<──QH #1 │         │10 SER<──QH #2  │
                 │11 D0          │         │11 D0          │         │11 D0          │
                 │12 D1          │         │12 D1          │         │12 D1          │

                 └────────────────┘        └────────────────┘        └────────────────┘
                 100nF VCC-GND             100nF VCC-GND             100nF VCC-GND

  All VCC pins ──── 3.3V
  All GND pins ──── GND
  All SH/LD pins tied together ──── Daisy D8
  All CLK pins tied together ──── Daisy D6
  All CLK INH (pin 15) ──── GND (clock always enabled)

  SER chain: #1 SER ── GND (first in chain)
             #1 QH  ──> #2 SER
             #2 QH  ──> #3 SER
             #3 QH  ──  not connected (end of chain)

  Daisy D7 reads from #1 QH (data shifts out #3 -> #2 -> #1)

BUTTON WIRING (each of 24 latching buttons):
  3.3V
   │
   [latching pushbutton]
   │
   ├──> 74HC165 parallel input (D0-D7)
   │
  [10k]  (pulldown)
   │
  GND

BUTTON TO SHIFT REGISTER MAPPING:
  IC #1: D0-D7 = Gate toggles for steps 1-8, voice A
  IC #2: D0-D7 = Gate toggles for steps 1-8, voice B
  IC #3: D0-D7 = Gate toggles for steps 1-8, voice C

READ SEQUENCE (firmware):
  1. Pull SH/LD LOW (load parallel data)
  2. Pull SH/LD HIGH
  3. Read D7 (first bit from IC #1 D7)
  4. Pulse CLK, read D7 (next bit)
  5. Repeat 24 times total for all 3 bytes
```

```
                     ┌───────────┐
  MCP4728 VOUTx      │           │
       │              │  TL074    │
      [10k] R_in      │  (1/4)   │
```

## 10. Gate Outputs (3 channels)

```
GATE A OUTPUT (repeat for Gate B on D18, Gate C on D19):

  +5V (from Eurorack power)
   │
  [1k]  R (pullup)
   │
   ├──────────────────> GATE A JACK TIP
   │
   D  (drain)
   │
  [2N7002]  N-ch MOSFET (SOT-23)
   │
   G ──── Daisy D17
   │
   S  (source)
   │
  GND

  MOSFET pinout (SOT-23):
    Pin 1: Gate
    Pin 2: Source
    Pin 3: Drain

  Daisy D17 HIGH (3.3V) --> MOSFET ON --> output pulled to GND = 0V (gate off)
  Daisy D17 LOW (0V)    --> MOSFET OFF --> output pulled to +5V (gate on)

  NOTE: Logic is inverted in hardware. Firmware must invert:
    GPIO LOW  = gate ON  (+5V at jack)
    GPIO HIGH = gate OFF (0V at jack)

  Alternative (non-inverting): use P-channel MOSFET or NPN transistor.

  ACCENT ACCENT ── ACCENT FOR NON-INVERTED ACCENT VERSION WITH NPN:

  Daisy D17 ──[1k]── Base
                      │
                   [2N3904] NPN (SOT-23)
                      │
               Emitter ── GND
                      │
              Collector
                      │
                      ├──> GATE A JACK TIP
                      │
                     [1k]
                      │
                     +5V

  D17 HIGH --> NPN ON --> collector pulls low...

  Actually, for a clean +5V gate output, use this proven circuit:

  Daisy D17 ──[1k]──┬── Base [2N3904]
                     │         │
                   [10k]    Emitter ── GND
                     │
                    GND     Collector
                               │
                               ├──> GATE JACK TIP
                               │
                              [4.7k]
                               │
                              +5V

  D17 HIGH (3.3V) --> transistor ON --> collector LOW = ~0.2V
  D17 LOW  (0V)   --> transistor OFF --> collector HIGH = +5V via pullup

  This is still inverting. For simplest non-inverting output:

RECOMMENDED GATE CIRCUIT (non-inverting, 2-transistor):

  Daisy D17 ──[1k]──┬── Base [Q1: 2N3904]
                     │              │
                   [10k]         Emitter ── GND
                     │
                    GND          Collector
                                    │
                              ──[1k]── Base [Q2: 2N3904]
                                    │              │
                                  [10k]         Emitter ── GND
                                    │
                                   GND          Collector
                                                   │
                                                   ├──> GATE JACK TIP
                                                   │
                                                  [1k]
                                                   │
                                                  +5V

  D17 HIGH --> Q1 ON --> Q1 collector LOW --> Q2 OFF --> output HIGH (+5V) = GATE ON
  D17 LOW  --> Q1 OFF --> Q1 collector HIGH --> Q2 ON --> output LOW (~0.2V) = GATE OFF

  Non-inverting. Firmware writes HIGH for gate on, LOW for gate off.
  Use 6x 2N3904 (SOT-23) total for 3 gate channels.
  Or: use a 74HC04 hex inverter IC instead of discrete transistors (single IC, 6 inverters,
      use 3 pairs for double-inversion = non-inverting buffer with +5V level shift).

SIMPLEST RECOMMENDED APPROACH:
  Use a 74HCT04 hex inverter powered from +5V.
  HCT logic accepts 3.3V inputs as valid HIGH.
  Two inverters per channel = non-inverting buffer, output swings 0 to +5V.

                    74HCT04 (SOIC-14)
                 ┌────────────────┐
                 │14  VCC ── +5V  │
                 │ 7  GND ── GND  │
                 │                │
  Daisy D17 ──>  │ 1  1A   1Y  2 │──> │ 3  2A   2Y  4 │──> GATE A JACK
  Daisy D18 ──>  │ 5  3A   3Y  6 │──> │ 9  4A   4Y  8 │──> GATE B JACK
  Daisy D19 ──>  │11  5A   5Y 10 │──> │13  6A   6Y 12 │──> GATE C JACK
                 └────────────────┘
                 100nF VCC-GND

  Each gate channel uses 2 inverters (double inversion = buffer).
  3 channels x 2 inverters = 6 inverters = exactly one 74HCT04.
  Clean +5V output. Firmware: HIGH = gate on, LOW = gate off.
```
